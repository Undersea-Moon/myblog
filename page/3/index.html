什么是数据库

答：数据库是数据的”仓库“，根据数据结构（二维表）来组织，存储和管理数据的仓库

数据库常用术语

表 : 数据库中的数据以表的形式显示

行 : 表中的一个元组（字段），各个记录的名称，也可以理解为一条信息或是一个对象

列 : 记录名称所对应的数据域，也就是对表的描述，可以理解为属性

冗余 : 多余无用的数据，会降低系统的性能
